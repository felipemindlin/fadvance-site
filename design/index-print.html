Notas para la próxima ronda

Antes de avanzar.

01
Imágenes

Hero, tramas del About y logos de PAZ + City National Bank ya están integrados desde el brandbook. Si tienen una foto de portada distinta (skyline de Miami, edificio puntual del portfolio), la reemplazo en las 4 versiones.

02
Tipografía

V2 y V3 piden Helvetica Neue; donde no esté instalada, el browser cae a Helvetica → Inter (look casi idéntico). Para producción podemos licenciar o usar Inter como reemplazo permanente.

03
Contenido

Mantuve el texto del sitio archivado. El doc de Content dice "40 años de experiencia" — Wayback decía 25. Confirmar cuál usar. Si quieren agregar Track Record / Team / Press, se suman en la próxima ronda.

04
Siguiente paso

Elegir 1–2 direcciones para refinar. También se pueden mezclar — por ejemplo layout y estructura de V1 con tipografía editorial de V2, o aplicar la capa interactiva de V4 sobre V3.
